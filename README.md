### Course Philosophy: Concept Extension  
Rather than treating machine learning as a black box, this course emphasizes a **concept extension** approach:

- Start with **foundational statistical models** (e.g., OLS regression)
- Extend them using **machine learning techniques** (e.g., Lasso, regularization, cross-validation)
- Compare **causal interpretation** and **predictive performance**
- Understand trade-offs between **explainability and accuracy**

Through this approach, I am learning how to:
- Combine **causal inference** with **predictive analytics**
- Evaluate models using both **economic reasoning** and **statistical metrics**
- Apply machine learning responsibly in economic contexts
